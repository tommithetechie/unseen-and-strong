# Unseen and Strong

An Android app built with Jetpack Compose to support people living with chronic illness. Offline-first, empathetic, and built around the reality of managing invisible illness day-to-day.

## Phase 1 — The Cocoon MVP

- Daily Symptom & Spoon Tracker
- Flare Day Mode
- Digital Comfort Box
- Invisible Illness Journal & Unseen Wins Tracker
- Gentle Routine Builder
